# Modello3

Zaawansowany, responsywny landing page dla portfolio fotomodelki.

Struktura:
- index.html — główna strona
- styles.css — style
- script.js — galeria i lightbox
- assets/ — dodaj tutaj swoje zdjęcia (hero.jpg, thumb-1.jpg ... thumb-9.jpg)

Instrukcje:
1. Dodaj zdjęcia do katalogu assets
2. Otwórz index.html w przeglądarce lub hostuj na GitHub Pages
